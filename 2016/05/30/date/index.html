我们先回顾下

var num = new Number();//数字

var oDate = new Date(2015,5,30);//日期

var oDate = new Date();//创建一个日期对象

var newTime = new Date(2015,1,13,0,0,0);

alert(oDate);//console.log(oDate);

var oDate = new Date();//当前时间

var oDate = new Date(‘2015-6-29’);//此刻的时间哦

function show() {

alert(“函数”);

}

eval

eval()图像可将字符串转换为代码执行，并返回一个或多个值eval调用时，实例eval((“javascript代码”))

eval()的返回值遵循以下规则：

1 如果eval（）的参数不是字符串，那么eval（）将直接返回参数。

2 如果eval（）的参数是字符串，那么eval（）将这个字符串解析成代码后进行执行，并返回最后一行代码执行的结果。

3 如果字符串无法解析成合法的代码，eval（）将抛出错误。
